## 1. 产品概述

### 1.1 产品简介

鱼隐是一款 Windows 桌面应用，允许用户在办公场景下隐蔽地观看体育直播、视频内容以及阅读小说。应用以无框、透明、始终置顶的小窗口形式运行，不显示在任务栏中，并可一键伪装为常用办公软件界面（VS Code、Excel、系统更新、邮件客户端），从而在"老板路过"等场景下快速切换。

### 1.2 目标用户

- 办公场景下希望观看体育赛事直播的用户
- 需要在工作时间内隐蔽使用视频播放器的用户

### 1.3 核心价值

| 价值 | 说明 |
|------|------|
| 隐蔽性 | 无任务栏图标、始终置顶小窗、可调透明度、老板键一键隐藏 |
| 伪装性 | 多种高仿真办公软件主题，一键切换"工作模式" |
| 易用性 | 内置直播源与小说源、支持自定义 URL、窗口可拖拽缩放 |
| 兼容性 | 支持 HLS 直播流 (m3u8)、普通视频 (mp4/webm) 及网页播放 |
| 轻量化 | 单文件 Portable exe，无需安装，双击即用 |

## 4. 共享概念

### 4.1 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F8` | 老板键 | 瞬间隐藏/显示窗口 |
| `Ctrl + `` | 切换工作/观赛模式 | 在伪装主题和视频播放之间切换 |
| `Ctrl + Shift + ↑` | 增加透明度 | 每次增加 0.05，上限 1.0 |
| `Ctrl + Shift + ↓` | 降低透明度 | 每次减少 0.05，下限 0.05 |

快捷键在主进程中通过 `globalShortcut` 模块注册，窗口关闭和应用退出时自动注销。

### 4.2 系统托盘

- 托盘区域显示蓝色圆形图标（程序生成，不依赖外部文件）
- 左键点击：切换窗口显示/隐藏，显示时自动获取焦点
- 右键菜单："显示 / 隐藏"、"退出"

### 4.3 安全设计

| 措施 | 说明 |
|------|------|
| `contextIsolation: true` | 渲染进程与 preload 隔离 |
| `nodeIntegration: false` | 渲染进程无 Node.js 访问权限 |
| `contextBridge` API | 仅暴露必要的 IPC 方法到渲染进程 |
| Preload 白名单 | 渲染进程只能调用预定义的 `window.api` 方法 |

## 5. 构建与分发

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动 Vite 开发服务器 + Electron |
| `npm run build` | 编译 TypeScript + 构建 Vite |
| `npm run dist` | 构建并打包为 Portable 可执行文件 |

打包格式：Windows Portable（单文件 .exe，x64），应用 ID：`com.yuyin.app`，输出目录：`release/`。

### 5.1 发布流程

1. 更新代码后执行 `npm run dist` 生成新版本 exe
2. 提交代码到 GitHub：`git add -A` → `git commit` → `git push`
3. 在 GitHub Releases 页面创建新 tag（如 `v1.0.0`），上传 exe 和使用说明

### 5.2 国内打包环境

项目已配置 `.npmrc`，使用 npmmirror 镜像加速 Electron 下载：
```
ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
```

### 5.3 分发渠道

- **官网下载页**（面向普通用户）：https://yuyinmoyu.com
- **GitHub Releases**（开发者/备份）：https://github.com/FloraSun007/yu-yin/releases
- 官网由 GitHub Pages 托管，源文件为 `docs/` 目录，随仓库自动部署
- 用户下载 Portable exe 后双击运行，Windows 可能提示"未知发布者"（无代码签名证书）
- 发布包附带 `使用说明.txt`

## 6. 官网设计

官网由 GitHub Pages 托管，源文件为 `docs/` 目录，包含首页 (`index.html`) 和使用文档 (`help.html`)。

### 6.1 技术栈

- 纯 HTML + CSS + 原生 JavaScript，无框架依赖
- 字体：Inter（Google Fonts）
- 数据统计：百度统计（`hm.baidu.com`）
- 暗色主题：主背景 `#09090b`，主色调 indigo/violet 渐变

### 6.2 首页结构 (`index.html`)

| 区块 | 说明 |
|------|------|
| 导航栏 | 固定顶部，毛玻璃效果。左侧 Logo「鱼隐」，右侧三个链接：使用文档、价格、联系我们 |
| Hero | 渐变光晕背景，大标题「鱼隐」，副标题「上班摸鱼，理直气壮」，下载按钮（带 glow 动效），徽章：免安装/无广告/开源 |
| 功能特性 | 3×2 网格卡片，6 个特性：老板键、一键伪装、透明度、小窗口、隐身运行、多格式支持 |
| 使用场景 | 2×2 网格，CSS mockup 模拟窗口：体育直播、小说阅读、自由浏览、伪装界面 |
| FAQ | 6 个常见问题，卡片式排列。「真的免费吗？」回答「限时免费，无隐藏收费」 |
| 价格 | 单一价格卡片，¥0，绿色主题，列出所有功能特性 |
| 联系我们 | 两张卡片：使用文档（链接到 help.html）、我们是谁（触发关于弹窗） |
| Footer | 简洁文字排版：鱼隐 Logo + slogan「打工人的底气，藏在屏幕角落里」+ `Made with courage · YuYin`。无导航链接 |

#### 关键设计决策

- **不展示 GitHub 仓库链接**：全站无 GitHub repo 显示链接（导航栏、Footer、FAQ 等均无仓库入口）
- **收费文案**：使用「限时免费」替代「完全免费/永久免费」，为未来收费预留空间
- **下载链接**：直接指向 GitHub Releases 的 exe 文件，通过百度统计追踪下载事件
