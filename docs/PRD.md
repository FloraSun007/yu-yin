## 1. 产品概述

### 1.1 产品简介

鱼隐是一款 Windows 桌面应用，允许用户在办公场景下隐蔽地观看体育直播、视频内容以及阅读小说。应用以无框、透明、始终置顶的小窗口形式运行，不显示在任务栏中，并可一键伪装为常用办公软件界面（VS Code、Excel、系统更新、邮件客户端），从而在"老板路过"等场景下快速切换。

### 1.2 目标用户

- 办公场景下希望观看体育赛事直播的用户
- 需要在工作时间内隐蔽使用视频播放器的用户

### 1.3 核心价值

| 价值 | 说明 |
|------|------|
| 隐蔽性 | 无任务栏图标、始终置顶小窗、可调透明度、老板键一键隐藏 |
| 伪装性 | 多种高仿真办公软件主题，一键切换"工作模式" |
| 易用性 | 内置直播源、支持自定义 URL、窗口可拖拽缩放 |
| 兼容性 | 支持 HLS 直播流 (m3u8)、普通视频 (mp4/webm) 及网页播放 |

## 4. 共享概念

### 4.1 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F8` | 老板键 | 瞬间隐藏/显示窗口 |
| `Ctrl + `` | 切换工作/观赛模式 | 在伪装主题和视频播放之间切换 |
| `Ctrl + Shift + ↑` | 增加透明度 | 每次增加 0.05，上限 1.0 |
| `Ctrl + Shift + ↓` | 降低透明度 | 每次减少 0.05，下限 0.05 |

快捷键在主进程中通过 `globalShortcut` 模块注册，窗口关闭和应用退出时自动注销。

### 4.2 系统托盘

- 托盘区域显示蓝色圆形图标（程序生成，不依赖外部文件）
- 左键点击：切换窗口显示/隐藏，显示时自动获取焦点
- 右键菜单："显示 / 隐藏"、"退出"

### 4.3 安全设计

| 措施 | 说明 |
|------|------|
| `contextIsolation: true` | 渲染进程与 preload 隔离 |
| `nodeIntegration: false` | 渲染进程无 Node.js 访问权限 |
| `contextBridge` API | 仅暴露必要的 IPC 方法到渲染进程 |
| Preload 白名单 | 渲染进程只能调用预定义的 `window.api` 方法 |

## 5. 构建与分发

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动 Vite 开发服务器 + Electron |
| `npm run build` | 编译 TypeScript + 构建 Vite |
| `npm run dist` | 构建并打包为 Portable 可执行文件 |

打包格式：Windows Portable（单文件 .exe，x64），应用 ID：`com.yuyin.app`，输出目录：`release/`。
