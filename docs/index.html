常见问题

真的免费吗？

完全免费，无隐藏收费，无广告，开源项目。

会被 IT 部门发现吗？

鱼隐不在任务栏显示图标，窗口可调节透明度，还有老板键一键隐藏。

需要安装吗？

不需要。下载 exe 文件后双击即可运行，不写注册表，删除文件即卸载。

Windows 提示"未知发布者"？

点击"更多信息" → "仍要运行"即可。软件本身是安全的。

窗口找不到了？

按 F8 键切换显示/隐藏，或点击右下角系统托盘的蓝色圆形图标。

怎么完全退出？

右键点击右下角系统托盘图标 → 退出。
