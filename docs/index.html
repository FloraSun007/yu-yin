常见问题

Windows 提示"未知发布者"？

点击"更多信息" → "仍要运行"即可。这是因为应用没有购买代码签名证书，软件本身是安全的。

需要安装吗？

不需要。下载 exe 文件后双击即可运行，不写入注册表，删除文件即卸载。

窗口找不到了？

按 F8 键切换显示/隐藏，或点击右下角系统托盘的蓝色圆形图标。

怎么完全退出？

右键点击右下角系统托盘图标 → 退出。
