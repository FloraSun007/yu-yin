常见问题

你可能想知道的

真的免费吗？

完全免费，无隐藏收费，无广告，开源项目。不像某些摸鱼软件动辄收费几百块。

会被 IT 部门发现吗？

鱼隐不在任务栏显示图标，窗口可调节透明度，还有老板键一键隐藏。正常使用基本不可能被发现。

需要安装吗？

不需要。下载 exe 文件后双击即可运行，不写入注册表，不残留文件，删除 exe 即为卸载。

Windows 提示"未知发布者"？

点击"更多信息" → "仍要运行"即可。这是因为没有购买代码签名证书（要花钱），软件本身是安全的。

窗口找不到了？

按 F8 键切换显示/隐藏，或点击右下角系统托盘的蓝色圆形图标。

怎么完全退出？

右键点击右下角系统托盘图标 → 退出。
