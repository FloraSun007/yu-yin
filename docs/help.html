欢迎来到鱼隐

鱼隐是一款 Windows 桌面应用，专为办公场景设计——让你在工作时也能看直播、读小说、刷网页。

无论世界杯、NBA、英超，还是起点追更、摸鱼刷视频，鱼隐都能帮你隐蔽、安全、无痕地完成。

核心特点

  • 体育直播 — 内置 CCTV5 / CCTV5+ / CCTV16，世界杯、奥运会、NBA 一键观看
  • 视频播放 — 支持 m3u8 直播流、mp4/webm 视频文件，随意播放
  • 小说阅读 — 内置起点中文网、七猫免费小说，小窗追更不留痕
  • 网页浏览 — 输入任意网址，上网课、看股票、刷论坛
  • 伪装主题 — 一键切换 VS Code / Excel / 系统更新 / 邮件界面
  • 老板键 — 有人来了？一键隐藏，不留痕迹
  • 透明度调节 — 半透明窗口，背后有人也不怕
  • 完全免费 — 无广告、无收费、开源项目

下载与安装

系统要求

  • Windows 10 或 Windows 11（64 位）
  • 无需安装任何依赖

下载

前往 官网首页 点击「免费下载」按钮，或直接访问 GitHub Releases 下载最新版本。

安装（不需要安装）

鱼隐是 Portable（便携） 应用，下载 exe 文件后：

  1. 双击运行 — 不需要安装，直接启动
  2. Windows 可能提示"Windows 已保护你的电脑"，点击 「更多信息」 → 「仍要运行」
  3. 启动后，桌面会出现一个无框透明小窗
提示：鱼隐不会写入注册表，不会创建快捷方式。想卸载？直接删除 exe 文件即可。

首次启动

启动后你会看到鱼隐的主界面：

主界面截图 — 内置直播源列表
  • 左侧是直播源列表（CCTV5、CCTV5+、CCTV16）
  • 点击任意直播源即可开始观看
  • 也可以输入自定义 URL 播放其他直播流

体育直播

这是鱼隐最主要的功能。内置 CCTV5 / CCTV5+ / CCTV16 三大体育频道，世界杯、奥运会、NBA、英超、中超……想看什么看什么。

内置直播源

启动后在首页即可看到内置直播源：

直播源列表示意图
频道内容
CCTV5体育频道 — 足球、篮球、排球、乒乓球等综合体育赛事
CCTV5+体育赛事频道 — 更多样赛事直播，冰球、赛车、马拉松等
CCTV16奥林匹克频道 — 奥运相关赛事、纪录片

观看直播

  1. 点击直播源列表中的任意频道
  2. 视频加载后即可开始观看
  3. 窗口可拖拽移动，可调整大小
  4. 按 F8 一键隐藏窗口
世界杯 / 奥运会期间：CCTV5 和 CCTV5+ 会全天候直播赛事，打开鱼隐就不会错过任何一场比赛。

自定义直播源

除了内置频道，你还可以输入任何 HLS 直播流地址（m3u8 格式）：

  1. 在首页点击「自定义」或输入框
  2. 粘贴 m3u8 直播流地址
  3. 点击播放即可
支持的格式：HLS 直播流（.m3u8）、普通视频（.mp4 / .webm）、网页视频（任意 URL）。

视频播放

除了直播流，鱼隐也支持播放普通视频文件和网络视频。

播放网络视频

  1. 在首页输入框中粘贴视频 URL
  2. 支持 m3u8、mp4、webm 等格式
  3. 点击播放即可

播放控制

播放器支持基本的播放控制：播放/暂停、音量调节、进度拖拽（非直播流）、全屏切换。

窗口调整

  • 拖拽移动 — 按住窗口空白区域拖动
  • 缩放窗口 — 拖动窗口边缘调整大小
  • 透明度 — 按 Ctrl+Shift+↑ / Ctrl+Shift+↓ 调节

  2. 选择「起点中文网」或「七猫免费小说」
  3. 网页加载后即可在线阅读

自定义网址

你也可以输入其他小说网站的 URL，鱼隐会以网页形式打开，支持在线阅读。

网页浏览

鱼隐支持打开任意网址，相当于一个隐蔽的迷你浏览器。上网课、看股票、刷论坛，随你用。

使用方法

  1. 在首页输入框中输入网址
  2. 点击访问即可
  3. 网页会在鱼隐窗口内显示

常见用途

  • 看股票 — 打开东方财富、雪球等网站，实时关注行情
  • 上网课 — 在工作间隙学习，提升自己
  • 刷论坛 — 虎扑、知乎、B站，一个都不少
  • 看新闻 — 随时了解最新资讯
配合伪装主题：看股票时可以伪装成 Excel，上网课时可以伪装成 VS Code。别人远远看去，你正在"认真工作"。

伪装主题

鱼隐提供多种高仿真办公软件界面，一键切换"工作模式"。老板路过时，你的屏幕看起来就像在认真工作。

可用主题

主题说明
VS Code仿真 VS Code 编辑器界面，看起来在写代码
Excel仿真 Excel 表格界面，看起来在做报表
系统更新仿真 Windows 更新界面，看起来电脑正在更新
邮件仿真邮件客户端界面，看起来在收发邮件

切换主题

  1. 按 Ctrl + `（反引号键）在伪装主题和当前内容之间切换
  2. 再次按下 Ctrl + ` 返回之前的内容
快捷切换：熟练使用 Ctrl + ` 切换，可以在 0.1 秒内从看直播切换到"写代码"。

老板键与透明度

鱼隐的核心隐蔽功能，让你在任何情况下都不暴露。

老板键（F8）

按 F8 键，窗口会瞬间隐藏——不是最小化，是完全消失。任务栏看不到，Alt+Tab 也看不到。

  1. 按 F8 隐藏窗口
  2. 再按 F8 恢复窗口
  3. 也可以点击系统托盘（右下角）的蓝色圆形图标来切换

透明度调节

通过快捷键调节窗口透明度，即使有人站在背后也看不清你在看什么。

快捷键功能
Ctrl + Shift + ↑增加透明度（更清晰）
Ctrl + Shift + ↓降低透明度（更透明）
组合使用：半透明 + 小窗口 + 屏幕角落 = 完美摸鱼姿势。别人看你的屏幕，只会觉得有个不太清晰的窗口在跑什么东西。

窗口找不到了？

解决方法：按 F8 键切换显示/隐藏，或点击右下角系统托盘的蓝色圆形图标。

快捷键一览

所有快捷键都可以在应用运行时全局使用，无论窗口是否获得焦点。

快捷键功能说明
F8 老板键 瞬间隐藏/显示窗口
Ctrl + ` 模式切换 在伪装主题和当前内容之间切换
Ctrl + Shift + ↑ 增加透明度 每次 +0.05，上限 1.0
Ctrl + Shift + ↓ 降低透明度 每次 -0.05，下限 0.05

托盘操作

操作说明
左键点击托盘图标显示/隐藏窗口
右键点击托盘图标打开菜单：显示/隐藏、退出

常见问题

真的免费吗？
完全免费，无隐藏收费，无广告，开源项目。源码在 GitHub 上公开。
会被 IT 部门发现吗？
鱼隐不在任务栏显示图标，窗口可调节透明度，还有老板键一键隐藏。只要你自己不暴露，别人很难发现。
需要安装吗？
不需要。下载 exe 文件后双击即可运行，不写注册表，不创建快捷方式，删除文件即卸载。
Windows 提示"未知发布者"怎么办？
这是因为软件没有购买代码签名证书。点击「更多信息」→「仍要运行」即可。软件本身是安全的，源码公开可查。
窗口找不到了怎么办？
按 F8 键切换显示/隐藏，或点击右下角系统托盘的蓝色圆形图标。
怎么完全退出？
右键点击右下角系统托盘图标 → 退出。
直播源看不了怎么办？
直播源依赖 CCTV 官方服务器，偶尔可能因为网络波动导致加载失败。请稍后重试，或检查网络连接。
支持 Mac 吗？
目前仅支持 Windows 10/11。Mac 版本暂无计划。
数据安全吗？
鱼隐不收集任何用户数据，不联网上传任何信息。应用本身使用 Electron 安全沙箱架构，渲染进程无 Node.js 访问权限。
