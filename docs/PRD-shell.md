## 1. 模块概述

外壳（Shell）模块负责应用窗口的创建与管理、全局快捷键、系统托盘、整体 UI 布局以及安全与构建配置。它是所有功能模块（直播视频、伪装主题等）的承载容器。

## 2. 窗口系统

### 2.1 窗口属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 默认尺寸 | 600 × 450 px | 启动时窗口大小 |
| 最小尺寸 | 320 × 240 px | 防止窗口过小导致 UI 不可用 |
| 窗口边框 | 无 (`frame: false`) | 自定义标题栏 |
| 透明背景 | 是 (`transparent: true`) | 圆角窗口效果 |
| 始终置顶 | 是 (`alwaysOnTop: true`) | 覆盖在其他窗口之上 |
| 任务栏显示 | 否 (`skipTaskbar: true`) | 不在任务栏出现图标 |
| 背景色 | 完全透明 (`#00000000`) | 配合圆角 |

### 2.2 窗口拖拽

采用 JS 轮询方式实现窗口拖拽（非 CSS `-webkit-app-region: drag`），避免与按钮点击、悬停事件冲突。

- 用户在顶部栏空白区域按下鼠标时，记录鼠标与窗口的偏移量
- 以 16ms 间隔轮询鼠标屏幕坐标，使用 `setBounds()` 同步窗口位置并锁定尺寸不变
- 全局 `mouseup` 事件停止拖拽

### 2.3 窗口尺寸预设

提供三档快速尺寸切换按钮（仅播放状态下可见）：

| 预设 | 尺寸 |
|------|------|
| 小窗 | 400 × 300 px |
| 中窗 | 600 × 450 px |
| 大窗 | 800 × 600 px |

### 2.4 窗口透明度

- 通过快捷键精细调节：每次增减 0.05，范围 0.05–1.0
- 左侧边缘悬停感应区：鼠标进入时透明度临时提升 +0.3，离开后延迟 3 秒恢复

## 3. 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F8` | 老板键 | 瞬间隐藏/显示窗口 |
| `Ctrl + `` | 切换工作/观赛模式 | 在伪装主题和视频播放之间切换 |
| `Ctrl + Shift + ↑` | 增加透明度 | 每次增加 0.05，上限 1.0 |
| `Ctrl + Shift + ↓` | 降低透明度 | 每次减少 0.05，下限 0.05 |

快捷键在主进程中通过 `globalShortcut` 模块注册，在窗口关闭和应用退出时自动注销。

## 4. 系统托盘

- 应用启动后在系统托盘区域显示蓝色圆形图标（程序生成，不依赖外部文件）
- 左键点击托盘图标：切换窗口显示/隐藏，显示时自动获取焦点
- 右键菜单：
  - "显示 / 隐藏" — 切换窗口可见性
  - "退出" — 关闭应用

## 5. 用户界面布局

### 5.1 整体布局

```
┌─────────────────────────────────────┐
│  模块 Tab 栏（无活跃内容时显示）       │
│  [直播]  [小说]                      │
├─────────────────────────────────────┤
│  顶部栏 (32px)                       │
│  [鱼隐 / ←返回]  [拖拽区域]  [按钮组] │
├─────────────────────────────────────┤
│                                     │
│  主内容区                            │
│  - 直播源列表（未播放时）              │
│  - 视频播放器（播放时）               │
│  - 小说源列表 / 小说阅读（小说模块）   │
│                                     │
├─────────────────────────────────────┤
│  控制栏（播放时显示）                  │
```

### 5.2 顶部栏

- 高度 32px，半透明深色背景 + 毛玻璃效果
- **左侧**：
  - 源列表页显示"鱼隐"Logo（蓝色粗体）
  - 播放页显示返回按钮（鼠标移入顶部栏时出现）
  - 工作模式时显示"工作中"绿色徽章
- **中间**：拖拽区域（cursor: grab），mousedown 触发窗口拖拽
- **右侧**：按钮组（默认隐藏，鼠标移入顶部栏时显示）
  - 工作模式时：主题选择按钮组
  - 模式切换按钮（观赛 / 工作）
  - 最小化按钮（隐藏到托盘）
  - 关闭按钮（红色悬停效果）
- 双击顶部栏：最小化窗口

## 6. 进程间通信 (IPC)

渲染进程通过 `preload/contextBridge` 暴露的 `window.api` 与主进程通信：

| API 方法 | 方向 | 说明 |
|----------|------|------|
| `close()` | Renderer → Main | 关闭窗口 |
| `minimize()` | Renderer → Main | 隐藏窗口（最小化到托盘） |
| `setOpacity(value)` | Renderer → Main | 设置窗口透明度 (0.05–1) |
| `setSize(w, h)` | Renderer → Main | 设置窗口尺寸 |
| `startDrag()` | Renderer → Main | 开始窗口拖拽 |
| `stopDrag()` | Renderer → Main | 停止窗口拖拽 |
| `onToggleMode(cb)` | Main → Renderer | 监听模式切换快捷键 |
| `onOpacityChanged(cb)` | Main → Renderer | 监听透明度变化 |
| `pointsIsFirstLaunch()` | Renderer → Main | 检测是否首次启动 |
| `pointsInit(referralCode?)` | Renderer → Main | 初始化积分账户 |
| `pointsGetStatus()` | Renderer → Main | 获取积分状态 |
| `pointsStartConsume()` | Renderer → Main | 开始积分消耗 |
| `pointsStopConsume()` | Renderer → Main | 停止积分消耗 |
| `pointsPurchaseCreate(productId)` | Renderer → Main | 创建购买订单 |
| `pointsPurchaseStatus(tradeNo)` | Renderer → Main | 查询订单状态 |
| `pointsPurchaseClaim(tradeNo)` | Renderer → Main | 确认支付（用户自报） |

## 7. 安全设计

| 措施 | 说明 |
|------|------|
| `contextIsolation: true` | 渲染进程与 preload 隔离 |
| `nodeIntegration: false` | 渲染进程无 Node.js 访问权限 |
| `contextBridge` API | 仅暴露必要的 IPC 方法到渲染进程 |
| Preload 白名单 | 渲染进程只能调用预定义的 `window.api` 方法 |

## 8. 构建与分发

### 8.1 开发命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动 Vite 开发服务器 + Electron |
| `npm run build` | 编译 TypeScript + 构建 Vite |
| `npm run dist` | 构建并打包为 Portable 可执行文件 |

### 8.2 打包配置

- **打包格式**：Windows Portable ZIP（免安装压缩包）
- **目标架构**：x64
- **应用 ID**：`com.yuyin.app`
- **产品名称**：鱼隐
- **输出目录**：`release/`
- **应用图标**：`assets/icon.ico`（蓝色鱼形图标）
- **国内镜像**：`.npmrc` 配置 `ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/`
- **额外资源**：`icon.ico`、`alipay-qr.png` 通过 `extraResources` 打包到 `process.resourcesPath`

### 8.3 开发模式特性

- 连接 Vite Dev Server (`localhost:5173`) 实现热更新
- 连接失败时自动回退到构建产物
- 自动打开 DevTools（detach 模式）

### 8.4 自定义协议

注册 `local-assets://` 协议，用于在渲染进程中加载资源文件：

- 开发环境：从 `assets/` 目录读取
- 打包环境：从 `process.resourcesPath` 读取
- 用途：加载收款二维码图片等资源

### 8.5 桌面快捷方式

首次启动时（仅打包后），自动在桌面创建快捷方式：

- 快捷方式名称：鱼隐
- 图标：`process.resourcesPath/icon.ico`
- AppUserModelId：`com.yuyin.app`
- 已存在则跳过创建

## 9. 非功能需求

| 需求 | 指标 |
|------|------|
| 窗口拖拽延迟 | ≤ 16ms（60fps 轮询） |
| 老板键响应 | 即时隐藏，无动画延迟 |
| 主题切换延迟 | ≤ 300ms（CSS opacity 过渡） |
| 窗口最小尺寸 | 320 × 240 px |
| 透明度范围 | 5% – 100% |

## 10. 已知限制

1. **仅支持 Windows**：打包配置仅面向 Windows Portable (x64)
2. **无自动更新**：Portable 模式不支持自动更新，用户需手动下载新版本
3. **无代码签名**：未购买代码签名证书，Windows 会提示"未知发布者"
4. **解锁码硬编码**：主题解锁码写在前端代码中，非安全加密
5. **无后端服务**：所有数据仅存储在本地 localStorage
6. **预设源可用性**：第三方直播源可能随时失效
