## 1. 模块概述

伪装主题系统负责在用户触发"工作模式"时，在视频播放界面之上叠加一层高仿真的办公软件界面覆盖层，使应用外观迅速变为常见办公工具，从而规避注意。系统支持多种主题、主题切换动画、以及部分主题的解锁机制。

---

## 2. 主题列表

| 主题 ID | 名称 | 状态 | 说明 |
|---------|------|------|------|
| `vscode` | VS Code | 免费 | 高仿真 VS Code IDE 界面 |
| `excel` | Excel | 免费 | 高仿真 Excel 电子表格界面 |
| `system-update` | 系统更新 | 需解锁 | Windows 更新界面，含动画进度条 |
| `email` | 邮件 | 需解锁 | Outlook 风格邮件客户端界面 |

---

## 3. 主题切换行为

- **观赛模式（默认）**：正常显示视频播放界面
- **工作模式**：在视频播放界面之上叠加一层伪装主题覆盖层，覆盖层设置 `pointer-events: auto` 阻止底层视频交互
- 两种模式通过 `Ctrl + `` 快捷键或顶部栏按钮切换
- 工作模式下顶部栏出现主题选择器，可切换不同伪装主题
- 切换时使用 CSS `opacity` 过渡动画
- 工作模式下顶部栏左侧显示"工作中"绿色徽章

---

## 4. 主题解锁机制

- **免费主题**（`vscode`、`excel`）：默认可用，无需额外操作
- **锁定主题**（`system-update`、`email`）：需输入解锁码（`yuyin2026`）方可使用
- 解锁状态持久化至 `localStorage('yuyin-unlocked')`
- 解锁后自动选中该主题
- 免费主题始终包含在已解锁列表中，不受存储值影响

## 5. 主题内容规格

### 5.1 VS Code 主题 (`vscode`)

仿真 Visual Studio Code IDE 开发环境界面。

**组成区域：**

| 区域 | 内容 |
|------|------|
| 标题栏 | 菜单项（文件/编辑/查看/...）、窗口控制按钮 |
| 活动栏 | 资源管理器/搜索/Git/调试/扩展图标 |
| 侧边栏 | 文件树结构 |
| 编辑区 | 带行号的代码（React/TypeScript 语法高亮） |
| 状态栏 | Git 分支、错误数、行列号、编码格式 |

### 5.2 Excel 主题 (`excel`)

仿真 Microsoft Excel 电子表格界面。

**组成区域：**

| 区域 | 内容 |
|------|------|
| 标题栏 | 绿色主题，Excel 图标和文件名 |
| Ribbon | 选项卡（开始/插入/页面布局/公式/数据/审阅/视图）+ 工具按钮 |
| 公式栏 | fx 标识和公式输入 |
| 数据网格 | 10 行 × 8 列的中文销售数据表 |
| 底部 | 工作表标签 + 状态栏（平均值/计数/求和） |

### 5.3 系统更新主题 (`system-update`)

仿真 Windows 系统更新界面。

**组成区域：**

| 区域 | 内容 |
|------|------|
| 顶部 | Windows Logo |
| 标题 | "Windows 正在更新" |
| 动画 | CSS 关键帧进度条（5% → 95% 循环） |
| 百分比 | 动态百分比显示 |
| 底部 | 更新进度文字说明 |

### 5.4 邮件主题 (`email`)

仿真 Microsoft Outlook 邮件客户端界面。

**组成区域：**

| 区域 | 内容 |
|------|------|
| 标题栏 | Outlook 蓝色主题 |
| 工具栏 | 新建/回复/转发/删除/归档/标记 + 搜索框 |
| 文件夹列表 | 收件箱（含未读数）、已发送、草稿、已删除、垃圾邮件、归档 |
| 邮件列表 | 8 封邮件（含发件人、主题、时间） |
| 预览窗格 | 显示完整邮件内容 |

## 6. 数据持久化

| Key | 类型 | 说明 |
|-----|------|------|
| `yuyin-unlocked` | `ThemeId[]` | 已解锁的主题 ID 列表 |

免费主题 (`vscode`, `excel`) 始终可用，不依赖存储值。

---

## 7. 已知限制

1. **解锁码硬编码**：解锁码 `yuyin2026` 写在前端代码中，非安全加密
2. **静态内容**：所有主题内容为静态模拟，无实际交互功能
3. **主题数量有限**：当前仅 4 个主题，未来可扩展 PPT、Teams、Slack 等
